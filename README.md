## Basic Commands (Java)

Compile a program (say Program.java)

```sh
javac Program.java
```

```sh
# include jar lib
javac -cp .:example.jar Program.java
```

Execute a program

```sh
java Program
```

```sh
# include jar lib
java -cp .:example.jar Program
```
