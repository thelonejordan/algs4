### FAQ

How do I specify EOF to signal that standard input is empty? (Relevant to `RandomWord`)

On Mac OS X and Linux, type `Enter` `Ctrl-D`. On Windows, type `Enter` `Ctrl-Z` `Enter`, even in Git Bash.
